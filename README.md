## Dev Notes

### Dev Server
`node server`

Custom port:
`PORT=1234 node server`

Built version:
`node server build`

### Run Grunt build process
`grunt`

### Installing Bower Components
`bower install <thing> --save`

### Injecting Bower Components
`grunt bowerInstall`
